# Model Hosting Information

## ⚠ Important Notice

The trained model file (`silver_guard.onnx`) is **NOT hosted on GitHub**.

GitHub enforces a strict file size limit (100MB), and the ONNX model exceeds that limit.  
To maintain a clean and lightweight repository, all trained model artifacts are hosted externally.

---

## 🤖 Hugging Face Model Repository

The complete model files are available at:

👉 https://huggingface.co/tanishqmudaliar/SilverGuard

---

## 📦 Available Files

The following files are hosted on Hugging Face:

- `silver_guard.onnx`
- `vocab.txt`
- `model_config.json`

---

## 🗂️ How to Use

1. Download the required files from Hugging Face.
2. Place them inside the `model/` directory:

```text
model/
├── silver_guard.onnx
├── vocab.txt
└── model_config.json
```

3. Run inference normally.

---

## 🔎 Why External Hosting?

- Keeps GitHub repository lightweight
- Follows industry best practices for ML projects
- Ensures versioned model storage
- Supports large file handling
